## ğŸ“Œ Project Overview

The **Food Nutrition & Dietary Insights Dashboard** is an end-to-end Power BI project designed to analyze daily food consumption patterns and evaluate overall nutrition quality.

This dashboard enables users to:

- Understand nutritional composition of food items
- Identify high-risk foods (High Sugar, Sodium, Cholesterol, Calories)
- Compare meal-type calorie distribution
- Plan diet based on weight-loss or muscle-gain goals
- Make informed, data-driven dietary decisions

The project demonstrates real-world data modeling, DAX calculations, and interactive dashboard design applied to health analytics.

## ğŸ¯ Problem Statement

Many individuals consume food without fully understanding its nutritional impact. Hidden risks like high sodium or sugar intake can lead to long-term health issues.

- Converting raw nutritional data into clear KPIs
- Highlighting risk-based food classification
- Providing goal-oriented insights (Weight Loss / Muscle Gain)
- Enabling dynamic filtering by meal type and category
